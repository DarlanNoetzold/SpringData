# SpringData

## Projeto:
* Projeto de controle de funcionários, cargos e Unidade de Trabalho. Usando Java com Spring Data. 
* O objetivo desse projeto é aplicar os conhecimentos de Spring Data.

## Desenvolvimento:
* Uso do banco MariaDB.
* Java 1.8.
* Spring Framework.
